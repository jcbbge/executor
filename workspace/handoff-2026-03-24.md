## Architecture:
```
AGENTS_SCHEMA_PATH/skills/{skill-name}/SKILL.md
                    ↓
        createSkillLoader()
                    ↓
    executor.skill.load(name) → { name, metadata, content, path }
    executor.skill.list() → SkillMetadata[]
```

## Usage in Executor Runtime:
```typescript
// Load a skill's content
const skill = await tools.executor.skill.load({ name: "starting-session" });
console.log(skill.content); // markdown content after YAML frontmatter
console.log(skill.metadata.description); // from frontmatter

// List all skills
const skills = await tools.executor.skill.list();
```
